Restaurant Reviews App

This is a three-stage project for Udacity's Mobile Web Specialist Nanodegree, a Restaurant Reviews App. For stage 1 my task was to refactor the existing starter code so the app is responsive, accessible, and available offline using a service worker. For Stage 2 I will use server data instead of local memory, use IndexedDB to cache JSON responses, and increase performance. And in Stage 3 I will add a form so users can submit reviews, and optimize the site to meet stricter performance benchmarks.

Project Details

  • Responsive design
  • Accessible code
  • Available offline using a Service Worker
  • HTML5, CSS3, Flexbox
  • JavaScript and Grunt

See the Code

The app includes the main search page, and individual restaurant review pages generated from data stored in a corresponding JSON file.

Responsive Design

I made all content display on a range of media sizes utilizing available screen real estate. I also used responsive images sized appropriately depending on what viewport size the app is opened in.

Accessible Code

To make the code accessible to screen readers and users who tab through the page, I managed focus on the page so there is now a logical tab order, added a skip link, used the appropriate sematic elements in the HTML, and defined ARIA roles where needed. I updated the styling of the focus selector on buttons and links, and increased contrast between text and background where needed. I also added alternative text to images.

Offline Availability

Finally I added a service worker to cache responses to requests for site assets so that visited pages are rendered when there is no network access.
